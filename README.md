# Todo Backend App

A restful api for todo app

## Prerequest

Java11, mysql5.6, maven

## Installation

```bash
git clone git@github.com:kaanosman/spring-backend
cd spring-backend
mvn install
```

## Usage

```bash
change application.properties file
mvn spring-boot:run
```

## Api Documentation
http://localhost:8080/swagger-ui.html
